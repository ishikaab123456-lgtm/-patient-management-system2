# Patient Management System

## 📌 Project Description
This is a simple Patient Management System developed using Python Flask and CSV file storage. It helps to manage patient records and queue system in a hospital.

## 🚀 Features
- Add Patient
- View Patients
- Search Patient
- Update Patient
- Delete Patient
- Queue Management (Add & Serve)

## 🛠️ Technologies Used
- Python
- Flask
- HTML & CSS
- CSV (Database)

## 📂 How to Run
1. Install Flask
2. Run dsa.py
3. Open http://127.0.0.1:5000

## 📊 Data Structure Used
- List (for storing data)
- Queue (for patient waiting system)
